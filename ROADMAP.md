# ROADMAP.md

The phased build order. Each phase ends with a hard stop. The planner approves before the next phase begins.

## Status

- [x] **Kickoff** — Foundation, workspace, docs.
- [ ] **Phase A** — Schema design + Supabase setup.
- [ ] **Phase A.5** — Design system pass.
- [ ] **Phase B** — Auth + onboarding.
- [ ] **Phase C** — Listings (public). SSR + metadata + sitemap.
- [ ] **Phase D** — Seller dashboard + upgrade flow.
- [ ] **Phase E** — Search, filters, location.
- [ ] **Phase F** — Contact reveal (WhatsApp + phone, tier-gated server-side).
- [ ] **Phase G** — Pro tier upgrade via Paystack.
- [ ] **Phase H** — Escrow.
- [ ] **Phase I** — Admin, messaging, polish.
- [ ] **Phase J** — Launch readiness.

## Out of scope for v2 launch

- Native mobile app
- Multi-currency
- Bulk listing import
- Seller analytics beyond view counts
- Categories deeper than two levels
